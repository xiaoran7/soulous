# 用户手册

## 普通用户

1. 使用 `demo / demo123` 登录，或注册新账号。
2. 在“资料”页可维护昵称、邮箱和头像 URL，保存后工作台顶部会立即更新显示。
3. 在“任务”页创建学习任务，填写标题、课程、预计分钟和经验。
4. 可在任务列表中点击“编辑”调整任务信息，或点击“删除”移除不再需要的任务。已有提交、学习记录或成长记录的任务会被保留，避免误删审核和成长历史。
5. 点击“开始”将任务置为进行中。
6. 在下方选择任务并提交学习凭证：
   - 填写文字总结。
   - 可粘贴代码片段或资料链接。
   - 可主动上传截图凭证，上传前请确认不包含隐私信息。
7. 系统自动进行 AI 初审：
   - 通过：任务完成，宠物获得经验。
   - 需补充：补充更具体的学习过程后重新提交。
   - 打回：可重新提交或在“我的提交与申诉”中发起申诉。
8. 在“我的提交与申诉”中点击“查看反馈”，查看 AI 评分、原因、建议和关联任务。
9. 在“宠物”和“统计”页查看成长与学习趋势。
10. 打开“复盘”页生成每日总结，查看今日亮点、需要留意的问题、明日建议和宠物反馈。

## 课表

1. 打开“课表”页，顶部即是本周课程网格。
2. 导入：在教务系统导出学期课表（`.xls`）上传到底部“导入课表”，AI 会自动识别课程、教师、地点、节次与学期；也可展开“高级”粘贴课表 HTML。
3. 顶部可按“第 X 周”切换查看不同周次的课（按每门课的开课周次/单双周过滤）。点“设置开学日期”（第 1 周周一）后，表头会显示每天日期并能定位本周。
4. 点“添加课程”可手动补一节课（临时调课/活动占用）；点课程块查看详情并可删除。
5. 多学期时顶部出现学期下拉切换；点“管理课表”可按学期查看节数/清空，或清除未标注学期的课程。
6. 课表里的课程名会作为背景信息提供给“AI 拆解”，让 AI 更了解你的专业方向。

## AI 拆解

1. 打开“AI 拆解”。
2. 输入学习目标，例如“我要复习数据结构的栈和队列”。
3. 点击“生成任务”。
4. 将合适的任务加入计划。

## 每日复盘

1. 打开“复盘”页。
2. 系统会根据今日任务、提交、学习时长、经验日志和宠物状态生成总结。
3. 可点击“重新生成”刷新复盘内容。
4. 如果当天没有学习记录，复盘会提示从一个短任务开始。

## 管理员

1. 使用 `admin / admin123` 登录。
2. 打开“管理”页。
3. 查看提交队列并选择一条记录。
4. 根据 AI 分数、凭证内容和任务信息执行：
   - 人工通过：可填写复核意见，并调整实际发放经验。
   - 要求补充：填写用户需要补充的具体材料。
   - 驳回：填写驳回原因，便于用户重新提交或申诉。
5. 在“申诉队列”中处理用户申诉：
   - 通过申诉
   - 要求补充
   - 驳回申诉

## 凭证建议

高质量凭证应包含：

- 具体知识点
- 学习时长
- 练习结果
- 代码片段或链接
- 用户主动上传的截图 URL

不要上传包含聊天记录、账号密码、身份证、支付信息等隐私内容的截图。
