# 用户手册

## 普通用户

1. 未登录先看到落地页，点“开始使用”进入。**注册**：填用户名/密码/邮箱，点“发送验证码”收邮箱里的 6 位码填入再注册（邮箱必填）；**登录**用图形验证码。
2. 在“资料/设置”页可维护昵称、邮箱和头像 URL，保存后工作台顶部会立即更新显示。
3. 在“任务”页点右上角“新建任务”，填写标题、课程、预计分钟、类型等创建学习任务。
4. 任务以卡片列表展示，可按状态（待开始 / 进行中 / 审核中 / 待处理 / 已完成）和课程筛选。点任意任务卡片进入它的**详情页**——这个任务的全部操作都在详情页里完成。
5. 在详情页点“开始任务”置为进行中（进行中可“暂停 / 继续”）；未开始或被打回的任务可在详情页“编辑”或“删除”。已有提交、学习记录或成长记录的任务删除时会保留这些历史，避免误删审核和成长记录。
6. 任务进行中后，直接在详情页内提交学习凭证：
   - 填写文字总结。
   - 可粘贴代码片段或资料链接。
   - 可主动上传截图凭证，上传前请确认不包含隐私信息。
7. 系统自动进行 AI 初审：
   - 通过：任务完成，宠物获得经验。
   - 需补充：在详情页补充更具体的学习过程后重新提交。
   - 打回：在详情页修改凭证后重新提交，或在下方“提交记录”里发起申诉。
8. 在详情页的“提交记录”中点“查看反馈”，查看 AI 评分、原因、建议；对结果不满意可就地“发起申诉”。
9. 新账号默认没有宠物——去“宠物”页点“宠物市场”免费领养一只入门伙伴；之后可用金币购买更多品种、切换出战。每只宠物独立升级，升级解锁更多动作（满级 Lv30 全解锁）。
10. 在工作台点“每日签到”领金币和经验，连续签到天数越多奖励越高；“我的”页可看金币余额、连续打卡、宠物收藏与金币流水。
11. 打开“复盘”页生成每日总结，查看今日完成度、课程分布、近 7 天学习时长趋势，以及今日亮点、需要留意的问题、明日建议和宠物反馈。

## 课表

1. 打开“课表”页，顶部用“课表 / 考试安排 / 成绩”三个标签切换；课表标签是本周课程网格。
2. 同步：在底部“同步教务系统课表”输入学号 + 教务密码点“开始同步”，系统会自动登录教务系统拉取课表，并**一并抓回本学期考试安排和你各学期的成绩**（首次同步会装运行环境，耗时 1~2 分钟）。
3. 顶部可按“第 X 周”切换查看不同周次的课（按每门课的开课周次/单双周过滤）。点“设置开学日期”（第 1 周周一）后，表头会显示每天日期并能定位本周。
4. 点“添加课程”可手动补一节课（临时调课/活动占用）；点课程块查看详情并可删除。
5. 课表固定显示**本学期**（不再有“全部学期”混排）。顶部学期下拉为三个标签**共享**：切到某学期后课表/考试/成绩同时切换，旧学期单独查看。点“管理课表”可按学期查看节数/清空。
6. “考试安排”按学期列出每场考试的时间、考场、座位号；“成绩”按学期分组列出每门课成绩，并给出该学期门数 / 已修学分 / 加权平均绩点的小结。

## AI 拆解（对话）

像聊天软件一样跟 AI 学习助手对话，需要时让它把目标拆成一组可执行的学习任务。

1. 打开“AI 拆解”。左侧是可收起的侧边栏，对话按你自建的“分类”组织（未分类的进“默认”组）。
2. 在中间的输入框直接提问，或描述目标（如“帮我制定 3 个月通过日语 N4 的学习计划”）。
3. 想喂资料给 AI 时，点输入框左侧的“+ → 上传文件”，支持 Markdown / PDF / 纯文本，提取的文字会随消息一起发给 AI。
4. 当你让它“拆成任务/给我计划”时，AI 会给出一份可编辑的计划草案；逐条增删改后点“确认并创建任务”，任务就进入“任务”页。
5. 侧边栏可“发起新对话”、新建/重命名/删除分类、把对话移动到其他分类，或重命名/删除对话。

## 每日复盘

1. 打开“复盘”页。
2. 系统会根据今日任务、提交、学习时长、经验日志和宠物状态生成总结。
3. 可点击“重新生成”刷新复盘内容。
4. 如果当天没有学习记录，复盘会提示从一个短任务开始。

## 自习室

登录后默认就在自习室主页：整个应用的背景就是当前场景照片，中央玻璃卡切换**独享**/**共享**。

1. 独享：在中央卡写下今日目标（可选）、选正计时或倒计时时长，点“进入自习室”全屏沉浸专注。底部场景坞点缩略图即可换背景（全应用同步换），“声音氛围”调音乐/背景音，也可上传自定义场景图和音频。
2. 沉浸专注中：点左上角**场景名胶囊**可随时换场景；鼠标移到**屏幕顶部边缘**唤出导航条；点右下角宠物直达宠物页；倒计时归零自动结算。
3. 共享：在房间广场建房或加入房间后**直接进入沉浸态自习室**并自动开始专注，左上角面板实时显示房内成员的在线与专注时长。点「结束」结算专注并退出房间；自己建的房在广场可删除；空房和全员长时间离线的房间会被自动回收。

## 管理员

1. 管理员账号由部署时的引导配置创建（无默认 admin 账号），登录后打开“管理”页。
2. 查看提交队列并选择一条记录。
3. 根据 AI 分数、凭证内容和任务信息执行：
   - 人工通过：可填写复核意见，并调整实际发放经验。
   - 要求补充：填写用户需要补充的具体材料。
   - 驳回：填写驳回原因，便于用户重新提交或申诉。
4. 在“申诉队列”中处理用户申诉：
   - 通过申诉
   - 要求补充
   - 驳回申诉

## 凭证建议

高质量凭证应包含：

- 具体知识点
- 学习时长
- 练习结果
- 代码片段或链接
- 用户主动上传的截图 URL

不要上传包含聊天记录、账号密码、身份证、支付信息等隐私内容的截图。
